HTML é uma Linguagem de Programação?

O que é Linguagem de Programação?

É uma linguagem formal que funciona por meio de uma série de instruções, símbolos, palavras-chave, regras semânticas e sintáticas.

Permite que um programador crie programas a partir de um conjunto de ordens, ações consecutivas, dados e algoritmos.

O que é HTML?

HTML é a abreviação para a expressão inglesa HyperText Markup Language, que significa: "Linguagem de Marcação de Hipertexto".

É uma linguagem de marcação* utilizada na construção de páginas na Web. E não uma linguagem de programação como muitos pensam.

A tecnologia é fruto da junção entre os padrões HyTime e SGML.
